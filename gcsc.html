GCSC

GCSC, God's Call; Speedrunners Climb, was the first unity project that I had ever made, it was a demo for a fast paced, 3D platformer with superpowers! In the game demo you speedrun through levels to finish as fast as you can while utilizing the projectile system I made for the game.

In the game you can pick up a powerup from the ground, which allowed you to shoot red projectiles from your hands. These projectiles would explode when they connect with the environment. You could then use the force from the explosion to propel yourself to jump even higher and go even faster!

Since this was my time ever trying to create a game in unity, this was so far out of my comfort zone at the time and I was forced to learn and adapt so much to make this project come to fruition, however I'm so happy and proud I kept trying. This game ignited my desire to create games for my career!

During development for this project, I had to learn and implement:

  • Unity development- using built in unity tools.
  • C#- I had no prior knowledge of C# before this project however I knew python from my computer science GCSE.
  • Character movement script- I had to program and tweak character movement scripts including jumping and sliding.
  • Projectile development- to implement my projectile system I had to learn the raycast physics system and rigidbody physics.
  • Power up pickup and management.
  • Unity scene management.
  • UnityUI elements.
